Yunus Emre KIRBAYIR için modern, sade ve animasyonlu kişisel web sitesi.

## Geliştirme

Geliştirme sunucusunu başlat:

```bash
npm install
npm run dev
```

Tarayıcıdan `http://localhost:3000` adresini aç.

Ana sayfa: `src/app/page.tsx`

## SEO / Sitemap

Sitemap ve robots üretimi için (opsiyonel) site URL’ini tanımlayabilirsin:

```bash
export NEXT_PUBLIC_SITE_URL="https://senin-domainin.com"
```

## Build

```bash
npm run build
npm run start
```
